# Smartcook-project-management-plan
Dokumen Project Management Plan untuk SmartCook, aplikasi asisten memasak cerdas yang merekomendasikan resep berdasarkan bahan yang tersedia, menyediakan informasi nutrisi, dan terhubung dengan penjual sayur lokal. Repository ini berisi dokumen PMP lengkap, WBS, Gantt Chart, Risk Plan, serta Presentasi tugas besar Manajemen Proyek.
